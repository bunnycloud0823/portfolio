프로젝트 1. 고객 이탈 예측 모델 개발

온라인 쇼핑몰의 고객 이탈 현상을 사전에 감지하여 마케팅 전략을 최적화하기 위한 분류 모델을 구축했습니다. 50,000명의 고객 로그 데이터를 기반으로 구매 빈도, 클릭 수, 장바구니 이력, CS 접수 기록 등 다차원 변수를 결합했습니다.

주요 분석 결과, ‘최근 7일간 클릭 수 변화율’이 이탈 예측의 핵심 변수로 작용했습니다. 이를 바탕으로 쿠폰 발송 시점 최적화를 통해 캠페인 효율이 17% 향상되었습니다.

**기술 스택:** Python, Pandas, Scikit-learn, LightGBM, Optuna **성과:** AUC 0.91, 고객 유지율 78% → 86%

프로젝트 2. 부동산 시계열 예측 및 시각화

국토교통부 실거래가 데이터를 기반으로 아파트 매매가 추세를 예측했습니다. Prophet과 LSTM을 결합한 하이브리드 모델로 중장기 예측 정확도를 개선했습니다.

**핵심 기술:** Prophet, TensorFlow, Plotly **성과:** MAPE 4.8%, RMSE 0.039 **활용:** 공공데이터 포털 예측 서비스 시범 적용

프로젝트 3. SNS 감성 분석 및 키워드 트렌드

약 10만 건의 리뷰를 크롤링하여 KoBERT 기반 감성 분류 모델을 학습했습니다. 긍정 키워드와 부정 키워드의 상관도를 시각화하여 브랜드 이미지 변화를 추적했습니다.

**기술:** Python, PyTorch, KoBERT, Gensim **성과:** F1 0.93, 키워드 자동 군집화 97% 일치

프로젝트 4. 데이터 대시보드 자동화 시스템

Streamlit 기반 대시보드로 마케팅 데이터를 실시간으로 시각화하고, Google Sheets와 연동하여 주간 보고서를 자동 생성했습니다.

**성과:** 보고서 작성 시간 80% 단축, 전사 공유 시스템 구축 **기술:** Python, Plotly, Streamlit, Google API
